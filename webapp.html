Blazor Server

Run your app on the server with real-time UI updates via SignalR

  • ✓ Low initial download size
  • ✓ Full .NET runtime on server
  • ✓ Immediate access to server resources
  • ✓ Works on older browsers
  • ✓ Real-time updates with SignalR

Blazor WebAssembly

Run your app client-side in the browser using WebAssembly

  • ✓ No server required (static hosting)
  • ✓ Offline capability
  • ✓ Fast after initial load
  • ✓ Progressive Web App (PWA)
  • ✓ Reduced server load
